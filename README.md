# ActorLab

浏览器里的 **3D 导演台**:在实时 WebGL 场景中编排角色、道具、机位与全景,
逐关节摆姿,并捕捉镜头拼合分镜。交互与视觉细节还原 **libtv + 万兴 3D 导演台**。

**▶ 在线体验:[actorlab.pages.dev](https://actorlab.pages.dev/)**

![ActorLab](https://cdn.jsdelivr.net/gh/cdLab996/picture-lib/wudi/3d-characters/index.png)

引擎持有场景状态并向 UI 推送不可变快照,面板从快照渲染、再通过方法调用驱动引擎——
一套所见即所得的导演编排流程。

## 能力

| 能力 | 说明 |
| --- | --- |
| **场景搭建** | 添加角色(男 / 女 / 健硕 / 纤细 / 宽厚 / 儿童 / 少年 / 二头身共 8 种素体)、道具(方块 / 圆柱 / 球体 / 人体素模)、机位与群众阵列;支持本地上传 `.glb`。 |
| **群众阵列** | 按行数 / 列数 / 间距批量生成人群,一次布满背景。 |
| **姿势系统** | 全身多轴关节控制(身体、躯干、头部、手臂—肩、肘、腿—髋、膝,左右成对),配 20 组姿势预设(站立 / T 型 / 行走 / 跑步 / 坐姿 / 蹲下 / 单膝跪 / 双膝跪 / 叉腰 / 倚靠 / 鞠躬 / 思考 / 格斗 / 踢球 / 投掷 / 推进 / 招手 / 伸手 / 抱臂 / 看手机),可单个或全部复位。 |
| **机位调度** | 摄像机为一等实体,独立的 FOV 与注视目标;15+ 运镜预设按正面 / 侧背 / 俯仰 / 特殊分组(正面中景、正面特写、侧面跟拍、过肩、鸟瞰、荷兰角……);导演视角 / 机位视角一键切换,机位视角下带取景框与三分构图网格。 |
| **全景背景** | 内置图片 / 视频全景库(2:1、16:9、1:1、9:16、21:9 多种比例)或本地上传,等距柱状全景球支持天空颜色、水平旋转与球形半径调节。 |
| **截图分镜** | 在导演或机位视角按取景框裁剪导出画面,每台机位独立收集截图,可预览、下载、批量清空或发送到画布拼成分镜。 |
| **场景缓存** | 场景(实体、姿势、机位、场景设置、截图与全景背景)自动存入浏览器,下次访问即恢复;顶栏可手动保存或一键清缓存并重置为默认场景;空场景自动重建默认角色与机位。 |

## 界面

| 区域 | 内容 |
| --- | --- |
| **顶栏** | 「3D 导演台」标题 · 居中「导演视角 · 机位视角」切换 · 保存场景 · 清缓存重置 · 语言切换 · 明暗模式 · 帮助。 |
| **左栏「场景清单」** | 搜索 + 层级清单(机位 / 角色 / 道具,含图标、选中高亮、显隐 · 改名 · 删除);支持多选与右键批量菜单(打组 / 解组 / 显隐 / 副本 / 删除)。 |
| **视口** | 右上角坐标指示器 +「重置视角」;角色名牌;机位视角下叠加取景框与九宫格辅助线。 |
| **底部工具坞** | 变换(移动 / 旋转 / 缩放) · ＋添加(角色素体 / 群众阵列 / 几何模型 / 本地上传子菜单) · 全景图 · 添加机位(预设) · 画幅比例 · 截图 · 全屏。 |
| **右栏** | 未选中 →「3D 场景」面板(场景缩放 / 平移 / 旋转、全景背景与天空颜色、全景球水平旋转 · 半径、角色标签、地面显隐 / 透明度 / 高度);选中 →「角色 / 道具 / 摄像机」面板(名称 / 位置 / 旋转 / 统一缩放 / 颜色;角色另有「姿势」页;摄像机含 POV 预览、FOV、注视目标与截图管理)。 |

## 操作

| 操作 | 说明 |
| --- | --- |
| **环绕 / 缩放** | 空白处拖动环绕视角;滚轮缩放。 |
| **变换对象** | 选中对象后拖三色 gizmo 按当前模式变换(拖 gizmo 时镜头不跟转)。 |
| **键盘快捷键** | `V` / `R` / `S` 切换移动 / 旋转 / 缩放;`Del`(或 `Backspace`)删除选中。 |
| **添加 / 摆姿** | 底部工具坞「＋」添加角色 / 几何模型 / 群众,支持本地上传 `.glb`;右侧「姿势」面板逐关节摆全身。 |
